COMO INICIAR O PROJETO:
Tenha o banco de dados PostgreSQL instalado e rodando no seu computador.

Crie a base de dados com o nome projetospd no PostgreSQL.

Configure o usuário postgres com a senha 123456 (padrão do projeto).

Rode a classe Main para que as tabelas necessárias sejam criadas automaticamente no banco.

Execute a classe LoginAppView para abrir a interface gráfica da aplicação e acessar o sistema.
